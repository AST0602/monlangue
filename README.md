# Minimalist Language Tool 极简多语学习工具

一个基于 AI 驱动的、现代极简主义风格的多语翻译与生词收藏工具。

## 核心特性

- **极简设计**：采用 Apple 风格的高级灰配色（#F5F5F7），纯白卡片配合柔和阴影，提供极佳的阅读体验。
- **AI 驱动**：支持自定义 OpenAI 兼容接口（如 DeepSeek、GPT-4 等），实现中、英、法等多语种的地道翻译。
- **多语扩展**：代码采用配置驱动，可轻松扩展至日、德、西等更多语言。
- **语气定制**：内置“Formel”、“Standard”、“Oral”三种语气标签，让 AI 返回更符合场景的结果。
- **Notion 同步**：一键收藏生词至 Notion 数据库（或通过 Cloudflare KV 同步），支持自动打标签，方便后续复习。
- **响应式布局**：完美适配手机、平板及电脑端。

## 使用方法

1. **API 配置**：点击页面左下角的“设置”按钮，配置您的 AI API Key、Base URL 及模型名称。
2. **Notion 配置（可选）**：在设置中填入 Notion Token 和 Database ID，开启云端同步收藏功能。
3. **开始查询**：在顶部输入框输入任何单词或句子，按下回车即可获得多语翻译。
4. **语气切换**：点击输入框下方的胶囊按钮切换翻译风格。
5. **收藏生词**：点击卡片右上角的星星图标，将生词保存至本地及 Notion。

## 部署说明

本项目为单文件 HTML 设计，您可以：
- 直接在浏览器打开 `index.html` 使用。
- 部署至 GitHub Pages、Vercel 或 Netlify。
